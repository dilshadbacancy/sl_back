# Quick Reference: Auto-Docs

## One-Time Setup

```bash
# 1. Install husky (manages git hooks)
npm install husky --save-dev

# 2. Setup husky
npm exec husky install

# 3. Make pre-commit hook executable
chmod +x .husky/pre-commit

# Done! âœ¨
```

## Daily Usage

### Generate docs manually

## Troubleshooting

**Hook not running?**
```bash
chmod +x .husky/pre-commit
```

**Docs not generating?**
```bash
npm run generate-docs
# Check output for errors
```

**Want to disable temporarily?**
```bash
git config core.hooksPath ''
# To re-enable: git config core.hooksPath .husky
```
